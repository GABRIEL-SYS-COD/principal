📬 Formulario de Contacto Integrado con Google Sheets

Este formulario de contacto está integrado directamente con una hoja de cálculo de Google Sheets. Cuando una persona completa el formulario, sus datos se envían automáticamente a una hoja online, sin necesidad de servidores o bases de datos externas.

¿Para qué sirve?

  • ✅ Recoger mensajes o solicitudes directamente desde la web
  • ✅ Guardar automáticamente los datos en tu Google Sheets
  • ✅ Ideal para formularios de contacto, feedback, soporte, inscripciones y más

¿Cómo funciona?

  1. 🔧 1. Creás una hoja en Google Sheets y la nombrás “Respuestas”
  2. ⚙️ 2. Configurás un script en Apps Script que reciba los datos (como backend)
  3. 🌐 3. Publicás el script como “Web App” para obtener una URL pública
  4. 📩 4. El formulario HTML usa esa URL para enviar los datos vía fetch()
  5. 🗂️ 5. Los datos llegan automáticamente a tu hoja de cálculo en tiempo real

¿Qué incluye este componente?

  • 🖱️ Un botón flotante siempre visible en la esquina inferior derecha
  • 🧩 Un modal con animaciones que se abre automáticamente
  • 📥 Un formulario con nombre, email y mensaje
  • 🔒 Validaciones básicas y estado de envío (“Enviado”, “Error”...)
  • 📊 Integración directa con Google Sheets sin usar servidores

Este sistema es eficiente, seguro y completamente gratuito, usando solo herramientas de Google y HTML. Perfecto para desarrollos rápidos y sin complicaciones.
